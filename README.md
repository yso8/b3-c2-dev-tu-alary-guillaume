## Installation

Once the repository is cloned 

To start the program : 
```sh
py main.py 
```

To run the tests :
```sh
py -m tests.test_calculator.py
```

To use coverage :
```sh
py -m coverage run .\tests\test_calculator.py
py -m coverage html .\tests\test_calculator.py
```
